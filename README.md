# Python Pattern Printing

This repository contains beginner-friendly Python pattern printing programs using loops and nested loops.

## Patterns Included
- Square Star Pattern
- Right Triangle Pattern
- Reverse Triangle Pattern
- Number Triangle
- Same Number Triangle
- Floyd’s Triangle
- Alphabet Pattern

## Concepts Used
- for loop
- nested loops
- range()
- string multiplication
- pattern logic

## Technologies Used
- Python

## How to Run

```bash
python patterns.py
```

## Learning Concepts
- Loop control
- Nested iteration
- Pattern building
- Logic development
